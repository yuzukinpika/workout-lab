# 💪 筋トレ Lab

筋トレの記録・分析ができる、**単一HTMLファイル**のWebアプリです。
ビルド不要・外部ライブラリ不使用・完全オフラインで動作します。スマホのホーム画面に追加して使う想定です。

## 特徴

- **記録** — 種目ライブラリ（組み込み46種目＋自作種目）から選んで、重量×回数（＋任意でRPE）をサッと入力。前回の記録を自動表示。
- **履歴** — 日付別の一覧から、タップで編集・削除。
- **分析** — 習慣化（週目標リング・週連続達成ストリーク・バッジ）／活動カレンダー／ボリューム推移／部位バランス／種目別の推定1RM推移／RPE推移／トレーニング頻度／体重推移。
- **AIコーチ用エクスポート** — 推移データ＋アプリの自動観察メモをMarkdownファイルに出力。チャットAIに添付すれば費用ゼロで個別アドバイスが得られます。
- **筋トレに合った習慣化設計** — ストリークは「毎日連続」ではなく**週目標を達成した連続週数**でカウント。休養日で途切れません。

## 使い方

1. [`workout-app.html`](workout-app.html) をダウンロードしてブラウザで開くだけ。
2. スマホでは「ホーム画面に追加」するとアプリのように使えます。
3. 初めての方は「分析」タブ最下部の「デモデータを入れて試す」で全機能を確認できます。

### PWAとして使う（推奨）

GitHub Pages等のWebサーバーで配信すると **PWA** として動作します（`manifest.webmanifest`・`sw.js`・`icons/` を同じ場所に置くこと）。ホーム画面に追加すれば専用アイコン・全画面表示・オフライン起動に対応します。アプリを更新配信するときは `sw.js` の `CACHE` のバージョン番号を上げてください。

## データについて

- 記録はすべて**ブラウザの localStorage**（端末内）に保存されます。サーバーには何も送信されません。
- 「分析 → データ管理」からJSONでバックアップ／復元できます。機種変更時はこれを使ってください。

## 計算仕様（抜粋）

- **ボリューム** — バーベル/マシン＝重量×回数、ダンベル＝片手重量×2×回数、片側種目は両側で×2、自重＝体重×回数（体重は日付付きログから当時の値を使用）。
- **推定1RM** — Epley式とBrzycki式の平均（設定で切替可）。
- **週の定義** — 月曜始まり。

詳細な仕様・設計は [HANDOFF.md](HANDOFF.md) を参照してください。

## 技術

素の HTML / CSS / Vanilla JS のみ。グラフは自前のインラインSVGです。
